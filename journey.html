Assignment process

How this website came together.

This page explains the planning, design choices, and development steps behind the Novelle Interiors website.

Project path

From brief to finished website.

01

Understanding the Brief

The assignment asked for a modern luxury interior design website called Novelle Interiors. I identified the required pages, features, visual direction, and responsive behavior before starting the layout.

02

Choosing the Style

I used a warm neutral palette, elegant serif headings, clean sans-serif body text, large image-led sections, soft shadows, and generous whitespace to create a calm premium feeling.

03

Building the Structure

The website was built with separate HTML pages for Home, About, Services, Portfolio, Contact, and this Journey page. A shared stylesheet and JavaScript file keep the code organized and reusable.

04

Adding Interaction

JavaScript powers the sticky navigation effect, mobile menu, theme toggle, testimonial carousel, portfolio filtering, expandable project details, and subtle scroll animations.

05

Preparing for Deployment

The final files are static and ready for GitHub Pages. The most important deployment detail is keeping index.html at the top level of the repository with the css and js folders beside it.
